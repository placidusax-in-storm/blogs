## Bad

```jsx
created(){
    const tmpArr = [];
    for (let i = 0; i < originArr.length; i++) {
        if (predicate(this.originArr[i])) {
            tmpArr.push(this.originArr[i])
        }
    }
    this.arr = tmpArr;
}
```

## Good

```jsx
computed: {
    arr(){
        return this.originArr.filter(predicate)
    }
}
```
